# 福田観光栗園 防除計算ツール 作業メモ

## 目的

福田観光栗園のくり園で、T70Pを使う防除・散布作業の必要量をスマホで確認できるようにする。

当初はHTMLファイルをスマホへ送って使う案だったが、現場運用に合わないため、最終的にGitHub PagesでURLを開くWebアプリ方式へ変更した。

## 園地・機材の前提

- 園地：福田観光栗園
- 面積：約4ha
- 4ha = 400a = 10a x 40区画
- 使用機体：DJI AGRAS T70P
- ドローン散布前提
- 観光栗園なので、実務上は開園・収穫開始前に散布を終える方針

## 登録・標準条件として入れたもの

| 対象 | 種類 | 標準条件 |
|---|---|---|
| ジマンダイセン水和剤 | 水和剤 | 10倍、8L/10a、収穫7日前まで、2回以内 |
| ディアナWDG | 水和剤 | 100倍、2L/10a、収穫前日まで、2回以内 |
| フェニックスフロアブル | 液剤 | 40倍、4L/10a、収穫前日まで、2回以内 |
| 塩化加里 | 粒剤 | 20kg/10a、希釈なし、水なし |

フェニックスは登録上 2〜4L/10a だが、後半防除・残効期待の初期値として 4L/10a にしている。

## 主な計算

### ジマンダイセン水和剤

- 4haで完成薬液320L
- 薬剤量32kg
- 水の目安288L
- T70P 70Lタンクなら 70L x 4回 + 40L x 1回

### ディアナWDG

- 4haで完成薬液80L
- 薬剤量0.8kg

### フェニックスフロアブル

- 4haで完成薬液160L
- 薬剤量4L

### 塩化加里

- 20kg/10a
- 4ha = 40区画なので合計800kg
- 20kg袋なら40袋
- 1m2あたり20g
- 粒剤のため、散布液量・希釈倍率・タンク容量は非表示

## UI変更の経緯

1. 最初は薬剤選択をプルダウンにした。
2. スマホで薬剤を切り替えても希釈倍率が反映されない問題が出た。
3. プルダウンをやめ、薬剤ごとの大きなボタンに変更した。
4. ローカルHTMLをスマホに送る方式は運用に合わないと判断し、GitHub PagesのWebアプリ方式へ変更した。
5. PWAキャッシュを追加したが、更新反映で古い表示が残る問題があったため、CSS/JSに `?v=` を付けてキャッシュを更新しやすくした。
6. 塩化加里追加後、粒剤であることが画面上分かりにくかったため、表示文言を「塩化加里（粒剤）」「粒剤 / 20kg/10a / 希釈なし / 水なし」に修正した。
7. 粒剤選択時に液剤用入力が残る問題があり、散布液量・希釈倍率・タンク容量を確実に非表示に修正した。

## 運用上の注意

- 農薬の登録内容、使用時期、使用回数、適用条件は散布前にラベル、JA、農薬登録情報で最終確認する。
- ジマンダイセン水和剤は粉剤で沈殿しやすいため、水を先に入れ、薬剤を少しずつ入れ、撹拌しながら調製する。
- 散布液の作り置きは避け、当日使い切る分を調製する。
- 初回T70P散布では、水だけで試運転してから薬液散布に入る。
- ドローン納品日には、初期設定、動作確認、散布設定、ノズル・ポンプ・タンク確認、バッテリー確認、操作説明を業者へ確認する。
