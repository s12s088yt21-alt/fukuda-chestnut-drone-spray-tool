# 福田観光栗園 防除計算ツール

福田観光栗園のくり防除で、DJI AGRAS T70Pを使ったドローン散布量を計算するための小さな静的ツールです。

## できること

- 薬剤・肥料選択に応じた標準散布条件の自動入力
- 面積、タンク容量から完成薬液量、薬剤量、水量、タンク回数を計算
- 塩化加里など粒剤の10aあたり散布量、総量、袋数を計算
- 1m2あたり、10aあたりの散布量を表示
- 時間計算タブで実散布時間、補給込み作業時間を試算
- スマホ表示対応
- GitHub PagesでURLから利用
- PWAキャッシュにより、一度開いた後は通信が弱い場所でも再表示しやすい

## 対象薬剤

- ジマンダイセン水和剤
- ディアナWDG
- フェニックスフロアブル
- 塩化加里

## 使い方

スマホではGitHub PagesのURLを開いて使います。

```text
https://s12s088yt21-alt.github.io/fukuda-chestnut-drone-spray-tool/
```

Safariで開いた後、共有ボタンから「ホーム画面に追加」しておくと、次回からアプリのように開けます。

初回表示には通信が必要です。一度開いた後はブラウザにキャッシュされ、通信が弱い場所でも再表示しやすくなります。
