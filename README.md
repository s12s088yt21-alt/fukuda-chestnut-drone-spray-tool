# 福田観光栗園 防除計算ツール

福田観光栗園のくり防除で、DJI AGRAS T70Pを使ったドローン散布量を計算するための小さな静的ツールです。

## できること

- 薬剤選択に応じた標準の希釈倍率・散布液量の自動入力
- 面積、タンク容量から完成薬液量、薬剤量、水量、タンク回数を計算
- 1m2あたり、10aあたりの散布量を表示
- 時間計算タブで実散布時間、補給込み作業時間を試算
- スマホ表示対応
- 単体HTML版でオフライン利用

## 対象薬剤

- ジマンダイセン水和剤
- ディアナWDG
- フェニックスフロアブル

## 使い方

通常版は `index.html` を開きます。

オフラインでスマホ利用する場合は、`福田観光栗園_防除計算ツール_スマホ版_v2.html` をスマホへ保存して開きます。このファイルはCSSとJavaScriptを内包しているため、通信環境がなくても動きます。

## 単体HTML版の再生成

```bash
node build-offline.js
```

## 注意

農薬の登録内容、使用時期、使用回数、適用条件は、散布前に必ずラベル、JA、農薬登録情報で最終確認してください。
